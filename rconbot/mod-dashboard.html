User Dashboard

Stores users from the r/Conservative server who either join r/Conservative or message the bot (in the case that a user's information is not readily available).

User Tickets

Lists all user tickets, open and closed. A ticket is an exchange between a user and the r/Conservative mods. When a user messages the Mod Mail bot, a ticket is created. Tickets can be closed either via a mod command, or via the web based dashboard.
